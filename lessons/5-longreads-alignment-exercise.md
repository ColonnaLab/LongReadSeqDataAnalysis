### **3. Run minimap2**

`minimap2` can write SAM output directly. SAM is human-readable but large, so we usually convert it immediately to BAM.

First, check the help page:

```bash
user1@vm-corso-colonna:~/lr-working$ minimap2 --help | less
```

Run the alignment:

```bash
user1@vm-corso-colonna:~/lr-working$ minimap2 \
  -ax map-ont \
  ../data-longreads/reference/KUN1163_reference.fasta \
  ../data-longreads/fastq/DRR187567.fastq.gz \
  > sam/DRR187567.KUN1163.sam
```

```diff
+ -a: output SAM format
+ -x map-ont: preset for Oxford Nanopore genomic reads
+ reference FASTA comes before reads FASTQ
+ > redirects the SAM output to a file
```

Inspect the SAM file:

```bash

user1@vm-corso-colonna:~/lr-working$ head -5 sam/DRR187567.KUN1163.sam
```

### **4. Convert SAM to Sorted BAM**

SAM files are useful for inspection, but BAM files are smaller and better for downstream analysis.

Convert SAM to BAM:

```bash
user1@vm-corso-colonna:~/lr-working$ samtools view \
  -b \
  sam/DRR187567.KUN1163.sam \
  > bam/DRR187567.KUN1163.bam
```

Sort the BAM file by genomic coordinates:

```bash
user1@vm-corso-colonna:~/lr-working$ samtools sort \
  -o bam/DRR187567.KUN1163.sorted.bam \
  bam/DRR187567.KUN1163.bam
```

Index the sorted BAM file:

```bash
user1@vm-corso-colonna:~/lr-working$ samtools index bam/DRR187567.KUN1163.sorted.bam
```

Check the output:

```bash
user1@vm-corso-colonna:~/lr-working$ ls -lh bam/
```

```diff
+ The sorted BAM stores alignments ordered by reference coordinates
+ The .bai index allows fast access to specific genome regions
```

### **5. Faster Alternative: Pipe minimap2 into samtools**

The previous steps are useful for learning, but they create a large intermediate SAM file. In real workflows, it is common to pipe `minimap2` directly into `samtools sort`.

```bash
user1@vm-corso-colonna:~/lr-working$ minimap2 \
  -ax map-ont \
  ../data-longreads/reference/KUN1163_reference.fasta \
  ../data-longreads/fastq/DRR187567.fastq.gz \
  | samtools sort \
      -o bam/DRR187567.KUN1163.minimap2.sorted.bam
```

```bash
user1@vm-corso-colonna:~/lr-working$ samtools index bam/DRR187567.KUN1163.minimap2.sorted.bam
```

```diff
+ The pipe symbol | sends minimap2 output directly to samtools sort
+ This avoids saving the large SAM file
```
